# CoreML-MaskRCNN
Convert the MaskRCNN torchscript model using the coremltool.

### Demo app

To run the demo app, simply do `pod install`

### Run the converter

Make sure you have PyTorch installed on your machine. 

```
> import torch
> torch.__version__ # '1.9.0'
```
Run the converter
```
> python converter.py
```
